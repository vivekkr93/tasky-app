
# Tasky App
Full-stack Task Management app using ASP.NET Core + EF Core + React.

## Backend
- Path: `backend/Tasky.Api`
- Run:
  ```bash
  cd backend/Tasky.Api
  dotnet restore
  dotnet ef database update
  dotnet run

## Frontend

 - Path: frontend/tasky-web
 - Run:
 ```bash
  cd frontend/tasky-web
  npm install
  npm run dev
